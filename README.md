# SPZ

A Swift library for loading and saving Gaussian Splat files (.spz format).

## Features

- Load and save .spz files (compressed Gaussian Splat format)
- Load and save .ply files (Gaussian Splat PLY format)
- Support for spherical harmonics up to degree 3
- Support for both legacy float16 and fixed-point position formats
- Memory-efficient packed data format
